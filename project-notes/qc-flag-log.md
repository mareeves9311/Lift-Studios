# Lift Studio QC Flag Log

This file is the running record of items flagged to the Quality Control agent, Megan's resolutions, and any rule updates triggered.

The Orchestrator checks this file during every system health check. Unresolved flags older than 48 hours are surfaced to Megan directly.

See `agents/quality_control.md` for the full QC flag format and process.
