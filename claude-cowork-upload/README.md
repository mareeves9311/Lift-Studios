# MR Studio Content Growth Kit - Claude Cowork Project

## Purpose

This folder is a clean handoff package for Claude/Cowork to help improve and execute the **MR Studio Content Growth Kit** outreach system.

MR Studio is a MR/service by Megan Reeves. The signature offer is the **Content Growth Kit**: a monthly website/social/content system for service-based and consumer-facing businesses.

## Main Goal

Create a polished, reusable outreach attachment that can be sent with a prospect-specific mini-audit.

The attachment should make a prospect think:

> "Wow, my business could look that polished? Hired."

## Current Brand Positioning

**MR Studio helps service-based and consumer-facing businesses turn their website and social media into a stronger first impression, with monthly content kits that make them easier to trust, book, visit, or buy from.**

## Current Direction

Use one fictional business example to demonstrate the full offer:

**Juniper Skin Studio**

A boutique skin studio / med spa example that shows:
- Brand/logo direction
- Brand kit
- Website homepage direction
- Social content examples
- UGC-style scripts/storyboards
- Monthly content bucket
- Service/pricing menu

## What Claude Should Do

Use the files in this folder to:

1. Critique the current offer and attachment strategy.
2. Improve the page-by-page copy.
3. Recommend the best tool workflow for Canva, Figma, and UGC/video tools.
4. Help create a production-ready version of the Juniper Skin Studio example.
5. Tighten the service/pricing menu.
6. Suggest what should be built now versus later.
7. Identify anything that feels too generic, too AI/template-like, or not premium enough.

## Key Files

- `PROJECT_BRIEF.md`: Full context, strategy, and current assets.
- `TASKS_FOR_CLAUDE.md`: Specific tasks and questions for Claude.
- `OUTREACH_ATTACHMENT_V2_COPY.md`: Current V2 page-by-page copy.
- `JUNIPER_SKIN_STUDIO_CREATIVE_BRIEF.md`: Fictional business details and creative direction.
- `TOOL_WORKFLOW_DECISIONS.md`: Current thinking on Canva, Figma, video tools, and Claude.
- `PROMPTS_TO_USE.md`: Prompts for creating website mockups, Canva pages, and UGC examples.
- `LINKS_AND_ASSETS.md`: Drive, Canva, tracker, and working links.

## Important Instruction

Do not make this feel like a generic agency deck. The final output should feel like a boutique studio portfolio piece that demonstrates specific, tangible outcomes.
