# Tool Workflow Decisions

## Current Recommendation

Use **Canva first** for the outreach attachment.

Use **Figma later** only if the homepage mockup needs to look more realistic or if the first outreach round gets traction.

Use **UGC/video tools later** after a prospect responds or asks for examples.

## Canva

Best for:

- Main outreach PDF
- Service/pricing menu
- Brand kit board
- Social mockups
- Simple homepage frame
- Fast PDF export

Why:

- Fastest path to a sendable asset
- Easy for Megan to edit
- Easy to duplicate for new niches
- Good enough for first outreach if manually polished

Watch-outs:

- Generated designs can feel template-like
- Homepage mockups can look less realistic than Figma
- Needs manual image/text cleanup before sending

## Figma

Best for:

- Realistic homepage mockup
- Design-system polish
- More credible website examples
- Later portfolio pieces

Use later if:

- The Canva homepage page looks too generic
- Prospects respond and ask for website examples
- Megan wants a stronger portfolio

Skip for now if:

- The goal is to send first outreach this week
- The homepage mockup can be represented well enough in Canva

## Claude / Claude Code

Best for:

- Copy critique
- Positioning refinement
- Page-by-page outline
- Fake business strategy
- UGC script ideas
- Website section copy
- Critiquing whether the offer feels premium

Not ideal for:

- Final visual layout
- PDF export
- Real brand asset production

## Claude.ai Artifacts

Best for:

- Quick rendered homepage concept
- HTML/CSS mockup that can be screenshotted
- Showing what the website could feel like

Use if:

- Canva homepage mockup is too flat
- A screenshot would make the attachment more convincing

## CapCut / Creatify / Predis.ai / Higgsfield

Best for:

- UGC-style video examples
- AI presenter/avatar concepts
- Short-form ad tests
- Video storyboards

Recommendation:

Do not create full videos for the first outreach attachment unless it is fast and easy.

For MVP:

- Show UGC hooks, scripts, and visual storyboards.
- Produce actual videos only after a prospect shows interest.

## Minimum Viable Sendable Asset

This week:

1. Canva PDF/report
2. One fictional business example: Juniper Skin Studio
3. Pricing/service menu
4. Brand kit page
5. Homepage mockup or homepage wireframe
6. Social/UGC examples
7. Monthly content bucket page

Later:

1. Figma homepage mockup
2. Actual UGC video samples
3. Niche-specific variants
4. Real case study
