# Audit to Claude Design Prompt: Template

Use this template to convert any completed client audit into a Claude Design brief.
Fill in the [brackets] with information from the audit document.

---

## How to Use

1. Complete the client audit (see audit_template.md)
2. Fill in this template using the audit findings
3. Paste the completed prompt into Claude Design
4. Generate Midjourney images using the style string at the bottom
5. Drop images into Claude Design slots

---
