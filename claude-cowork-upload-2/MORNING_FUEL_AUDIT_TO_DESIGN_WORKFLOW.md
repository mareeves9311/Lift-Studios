# Morning Fuel Audit To Design Workflow

## Recommended Flow

1. Run the diagnosis/audit in Claude.ai with web search enabled.
2. Bring the audit output back to Codex.
3. Codex turns the audit into a structured design brief and outreach package.
4. Canva or Claude Design creates the Morning Fuel-specific mockups.

## Why

The audit needs live access to:
- Instagram
- Google Business Profile
- Reviews/photos
- Website
- Menu
- Ordering path

Codex can access the website and organize the system, but Instagram is blocked from direct fetch here. Claude.ai or a browser-enabled research session is better for the diagnosis step.

## Audit Prompt

Use the file:

`morning_fuel_claude_audit_prompt.md`

## After Audit

Paste the full audit into Codex and ask:

> Turn this Morning Fuel audit into a design brief for a prospect-facing mockup attachment. Include brand/content direction, homepage recommendations, Instagram content examples, UGC scripts, and a first-month content plan.

## Output Codex Should Create

- Morning Fuel design brief
- Morning Fuel Canva attachment copy
- Morning Fuel UGC scripts
- Morning Fuel homepage mockup prompt
- Updated tracker notes
