## Current Thinking On Tools

### Canva

Best for:
- Outreach attachment
- Pricing/service menu page
- Brand board
- Social mockups
- PDF export

Pros:
- Fast
- Easy to edit
- Easy to send
- Good enough for first outreach

Cons:
- Website mockups can look less real/high-end than Figma
- Generated designs can feel template-like unless heavily edited

### Figma

Best for:
- High-end website homepage mockup
- More believable design portfolio pieces
- Brand board if I want more control

Pros:
- Stronger for real web design examples
- Better for layout precision
- More premium if done well

Cons:
- More time-consuming
- More skill/attention required
- Can become a rabbit hole before outreach starts

### Claude

Best for:
- Critiquing the offer
- Refining service copy
- Making the fictional business feel real
- Tightening pricing/menu language
- Reviewing whether the attachment feels premium and persuasive

Not best for:
- Final visual design production

### Creatify / Predis.ai / Higgsfield / CapCut

Potential uses:
- UGC-style video ad examples
- AI avatar/presenter videos
- Short-form ad concepts
- Product/service explainers
